# Gabriel-Gonzalez
En este repositorio comparto proyectos académicos y personales que abarcan el diseño de sistemas mecatrónicos, control, electrónica y programación.

Entre los proyectos destacados se encuentran una planta automatizada de mezcla y secado de material particulado implementada con PLC Siemens S7-1500 y Factory I/O, sistemas de comunicación entre microcontroladores mediante MQTT y RS-485, aplicaciones con STM32 y ESP32 para adquisición y procesamiento de datos, algoritmos de control y robótica en MATLAB/Simulink, así como investigaciones orientadas al uso de aleaciones con memoria de forma (SMA) para la modificación morfológica de álabes de aerogeneradores.
